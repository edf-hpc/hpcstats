HPCStats

HPCStats is a software to help generate statistics about the usage of HPC supercomputers.

It imports data from various sources (job schedulers, user directories, etc) to build a coherent PostgreSQL database with all the information to extract usefull statistics about users, jobs and supercomputer availability. Read the introduction for more details.
